FAQ Accordion using JavaScript

What is JavaScript?
JavaScript is a versatile programming language used to create dynamic and interactive web content.
Where is JavaScript used?
In web development (front-end and back-end), mobile apps, server-side applications, and more.
What are JavaScript data types?
Common types include Number, String, Boolean, Object, Array, Null, and Undefined.
What is the DOM?
The Document Object Model represents the structure of a web page, allowing JavaScript to interact with it.
What are Promises?
Objects used to handle asynchronous operations, providing .then() and .catch() for chaining.
What is closure?
A function that retains access to variables from its outer scope even after the outer function has executed.
